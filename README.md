# homebrew-tap

Homebrew tap for [adamorad](https://github.com/adamorad)'s tools.

## Usage

```bash
brew tap adamorad/tap
brew install --cask csv-quick-look
```

## Casks

| Cask | Description |
|---|---|
| [csv-quick-look](https://github.com/adamorad/csv-quick-look) | QuickLook extension for CSV and TSV files |
